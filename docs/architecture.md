# Architecture

Lena AI Team is designed as a personal AI department.

## Core idea

The system should not only parse content. It should transform market signals into practical decisions for Elena.

## Initial workflow

```text
Google Sheets source list
        ↓
Source loaders
        ↓
Platform watchers
        ↓
Scout analysis
        ↓
Telegram report
```

## Future workflow

```text
Scout → Strategist → Writer → Designer → Analyst
```

## Agent roles

### Scout
Collects market signals and identifies what matters for Elena.

### Strategist
Chooses the best content and offer opportunities.

### Writer
Creates text drafts in Elena's voice.

### Designer
Creates visual briefs, carousel structure, and image-generation prompts.

### Analyst
Studies what performs well across the market and, later, Elena's own content.
